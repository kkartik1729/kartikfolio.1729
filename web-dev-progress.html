My Web Development Journey

Follow my day-by-day progress as I learn and master web development skills

Day 1: Getting Started with VS Code

Setting Up My Development Environment

My journey into web development began with setting up Visual Studio Code, an incredibly powerful and customizable code editor. Here's what I accomplished today:

  • Installed VS Code and configured it for web development
  • Learned essential keyboard shortcuts to boost productivity
  • Set up extensions like Live Server, Prettier, and Auto Close Tag
  • Customized my workspace with themes and font settings
  • Created my first project folder structure

The most valuable shortcuts I learned today:

Ctrl+S: Save file
Ctrl+C/Ctrl+V: Copy/Paste
Alt+Shift+F: Format code
Ctrl+/: Toggle comment
Ctrl+` : Toggle terminal

Day 2: HTML Fundamentals

Learning Basic HTML Tags

Today I dove into HTML (HyperText Markup Language), the building blocks of every website. I focused on understanding the basic structure and essential tags:

  • Created my first HTML document with proper structure
  • Learned about headings (h1-h6), paragraphs, and text formatting
  • Added links using anchor tags and images using img tags
  • Created ordered and unordered lists
  • Added comments to improve code readability

Sample code snippet from today:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First Web Page</title>
</head>
<body>
  <h1>Hello World!</h1>
  <p>This is my first web page.</p>
</body>
</html>

Day 3: Advanced HTML - Forms and Tables

Creating Interactive Elements

On my third day, I expanded my HTML knowledge by learning how to create forms for user input and tables for organizing data:

  • Built forms with various input types (text, email, password, etc.)
  • Added dropdown menus, checkboxes, and radio buttons
  • Created submit buttons and form validations
  • Constructed tables with headers, rows, and columns
  • Used colspan and rowspan for complex table layouts

I created this simple contact form:

<form action="/submit" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" required>

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>

  <input type="submit" value="Submit">
</form>

Day 4: Introduction to CSS

Styling Web Pages

Today I started learning CSS (Cascading Style Sheets), which transforms plain HTML into visually appealing websites:

  • Explored the three ways to add CSS: inline, internal, and external
  • Learned about selectors, properties, and values
  • Styled text with font properties, colors, and alignments
  • Added background colors and images
  • Used margin and padding to control spacing

Sample CSS code I wrote today:

body {
  font-family: 'Arial', sans-serif;
  line-height: 1.6;
  margin: 0;
  padding: 0;
  background-color: #f4f4f4;
}

header {
  background: #0078ff;
  color: #ffffff;
  padding: 20px;
  text-align: center;
}

Day 5: CSS Box Model and Flexbox

Layout and Positioning

On my fifth day, I delved deeper into CSS by learning about layout models that are essential for creating modern, responsive designs:

  • Mastered the CSS Box Model (content, padding, border, margin)
  • Explored different display properties
  • Learned about positioning (static, relative, absolute, fixed)
  • Implemented Flexbox for creating flexible layouts
  • Created a simple responsive navigation bar

Flexbox code sample from today's practice:

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  flex-wrap: wrap;
}

.item {
  flex: 1 1 300px;
  margin: 10px;
  padding: 20px;
  background-color: #ffffff;
  box-shadow: 0 2px 5px rgba(0,0,0,0.1);
}
